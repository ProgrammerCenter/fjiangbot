### 启动方式
  1. 克隆这个项目`git clone https://github.com/liuqianpan2008/fjiangbot.git`
  2. 安装依赖 `npm i`
  3. 修改`src/config/.config.ts` 为 `config.ts`,并设置相关内容
  4. 运行`npm run app`
